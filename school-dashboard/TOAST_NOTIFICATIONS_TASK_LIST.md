## TOAST NOTIFICATION STANDARDS

### Success Messages
```javascript
toast.success("Action completed successfully!");
toast.success("Student added successfully!");
toast.success("Attendance marked for 25 students");
```

### Error Messages
```javascript
toast.error("Failed to save changes");
toast.error("Unable to delete: Item is in use");
toast.error("Network error. Please try again");
```

### Info Messages
```javascript
toast.info("Filter applied: Showing 10 results");
toast.info("Date changed to December 27, 2024");
```

### Warning Messages
```javascript
toast.warning("This action cannot be undone");
toast.warning("Approaching storage limit");
```

### Loading Messages
```javascript
const loadingToast = toast.loading("Processing...");
// After completion
toast.dismiss(loadingToast);
toast.success("Done!");
```
