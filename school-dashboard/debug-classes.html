Classes Data Debugger

Step 1: Check Browser Console

Open browser console (F12) and paste this:

// Check if classes data is loaded
const user = JSON.parse(localStorage.getItem('user') || sessionStorage.getItem('app_user') || '{}');
console.log('User:', user);

// Check React DevTools for AppContext
console.log('Look for AppContext in React DevTools');
console.log('Check these values:');
console.log('- classes array');
console.log('- classesWithTeachers array');
console.log('- loading state');
console.log('- error state');

Step 2: Check API Response

Open Network tab and look for:

  • /api/classes - Should return array of classes
  • Status should be 200
  • Response should have data

Step 3: Common Issues

  • No classes in database - Add classes first
  • API endpoint failing - Check backend is running
  • Authentication issue - Token expired
  • CORS issue - Check API URL configuration
