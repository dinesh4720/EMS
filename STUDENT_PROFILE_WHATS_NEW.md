## рҹҺЁ Color Coding System

### Reports Cards:
- **Blue** = Attendance (from-blue-50 to-blue-100)
- **Green/Success** = Fee Paid (from-success-50 to-success-100)
- **Yellow/Warning** = Fee Pending (from-warning-50 to-warning-100)
- **Green** = Parent App Active (from-green-50 to-green-100)

### Remarks:
- **Blue** = Academic remarks
- **Yellow/Warning** = Behavioral remarks
- **Red/Danger** = Medical remarks

### Academics:
- **Green** = A+ grades
- **Blue** = B+ grades
- **Orange** = C+ grades
