## Module Connection Summary

### Data Sources for Student Profile

1. **Students API** (`/api/students/:id`)
   - Personal information
   - Contact details
   - Parent information
   - Status

2. **Fees API** (`/api/fees/payments?studentId=:id`)
   - Payment history
   - Fee status
   - Pending dues

3. **Fees Structure API** (`/api/fees/structure/:classId`)
   - Fee breakdown
   - Installment schedule

4. **Attendance API** (`/api/attendance/:classId/:date`)
   - Daily attendance
   - Attendance percentage

5. **Classes API** (`/api/classes/:id`)
   - Class information
   - Class teacher
   - Subjects

6. **Staff API** (`/api/staff/:id`)
   - Class teacher details
   - Subject teachers

7. **❌ Missing APIs**
   - `/api/students/:id/remarks` - Teacher remarks
   - `/api/students/:id/documents` - Document management
   - `/api/students/:id/academic-records` - Exam results
   - `/api/students/:id/parent-app-status` - App status
   - `/api/students/:id/achievements` - Awards
